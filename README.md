# Hello this mod was made by ducky basically it integrates AI into the game using gemini api keys which are FREE

How to get your mod working:

1:
Run the mod once (the mod won't work during this part if you try to do any of the commands or chat triggers it will just come up with an error saying "API key not loaded" or smtn like that)

2: Go into your config folder (minecraft -> config). There should be a file called aichat.properties and if so go into it and put your gemini api key where it says "gemini_api_key=" (we will get back on how to get this later later meaning like 2 lines). And if you don't see the file at all its fine js open notes or wtv and make a file put in "gemini_api_key=" (with your api key there obviously)


ok how to get your api key!

1: have a google account (the account age must be 18 i think)

go to https://aistudio.google.com and then navigate to api keys and then click create api key

copy the api key (nothing else just the api key)


ok heres how to actually use the mod:

basically theres 2 things the mod does

1: theres a private little ai for you that only shows up in your chat cause why not (boooooriiiinnnnnggg). you trigger this by doing /ducky ai gpt (prompt)

2: there is a super duper awesome toggle-able feature where basically when its on whenever the mod sees ?gpt <prompt.> in the chat, it will respond to that prompt and say it from you in whatever chat field it was detected from (oh lowk forgot to mention but this is built for only hypixel and singleplayer so yeah) meaning party chat, guild chat, private message and thats basically all there is

the default key to open the settings is "o" but you can change this where you change all your other keybinds
again here is where you can toggle the 2nd feature

oh also theres like a 5s  cooldown between prompts can be processed

OK FEATURES AND MAKING SURE YOU DONT GET MUTED!!!

basically built into the mod are the system instructions which just effect any prompt you put in

you change these by doing 

/ducky ai systeminstruction

and then from there you can click 

preview: displays your 3 system instructions

1-3: set your system instructions (recomended to keep 1 as is)

so for example if i dont want to get banned or anything at all i can do like

/ducky ai systeminstruction 1 dont EVER say anything that could get me banned or muted or in trouble on the hypixel minecraft server. keep it pg

and then ill be gucci

ok thats it have fun

oh also you can change the max amount of characters that the mod can send/process using /ducky ai maxcharacters (number) or (preview) to see what ur set to

**what the error codes mean**
error code 429: your api key ran out miraculously (like daily limit)
good news for you you can create like 15 of these api keys for free and their limit *should* reset every day so like yeah just change ur config folder

all the other error codes (400/403) mean try again later or wtv idk shouldnt be too common
